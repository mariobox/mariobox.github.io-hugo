Stack

This site is a DIY project to learn how to create and deploy a simple website using a static site generator and other widely available web development tools.

I’m currently using a static site generator called Hugo. Hugo is made with Go. It takes content written in Markdown and blends it with a set of templates that combine HTML and Go code, producing a static site that I deploy to the public web via Github Pages.

For styling, I am using a minimalistic responsive css sheet called style.css that I put together taking snippets of code from different open source sheets and mixing them with my own input.

I keep track of changes to the site with Git, and store my source files in a Github remote repository.

This is the personal website of Mario Sanchez Carrion, a Miami-based product marketing manager and web development enthusiast, currently exploring the intersection of technology, publishing and branding. [More]
